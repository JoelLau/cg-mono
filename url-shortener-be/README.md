# README

## Usage

Install dependencies:
```shell
bundle install
```

Create Database
```shell
bundle exec rails db:create
```

Migrate Database
```shell
bundle exec rails db:migrate
```

## Development

Run Tests (rspec)
```shell
bundle exec rspec
```

Format All Files (rubocop)
```shell
bundle exec rubocop -A
```
